## Features

- 🕌 &nbsp;All 5 daily prayers plus Sunrise
- 🔢 &nbsp;Multiple calculation methods
- 📴 &nbsp;Fully offline prayer time calculation
- 🔔 &nbsp;Athan notifications
- ⏰ &nbsp;Pre-prayer reminders
- 🧭 &nbsp;Qibla compass
- 📿 &nbsp;Qada counter
- 📖 &nbsp;After-prayer Adhkar
- 🗓 &nbsp;Monthly prayer times table with Hijri dates
- 🪟 &nbsp;Home screen widget
- 🌗 &nbsp;Light and dark theme
- 🌐 &nbsp;Available in English, Arabic and Urdu

## Why Awqat?

Prayer apps should be simple, trustworthy, and unobtrusive. Somewhere along the way, many of them stopped being that — bloated with features nobody asked for, supported by ads, or quietly collecting more data than they should.

Awqat is the alternative. No ads, no tracking, no accounts, no server that knows where you are or when you pray. Just a well-built tool that does one thing and does it with care.
